Carrinho de Compras

Este projeto simula um sistema simples de carrinho de compras, onde é possível adicionar produtos, definir quantidades e visualizar o valor total em tempo real.

##  Funcionalidades

- Seleção de produtos
- Definição de quantidade
- Adição de itens ao carrinho
- Cálculo automático do valor total
- Limpeza do carrinho

##  Tecnologias utilizadas

- HTML
- CSS
- JavaScript

##  Sobre o projeto

O **frontend** deste projeto foi desenvolvido com base no curso da **Alura**.

Já a parte de **lógica e funcionamento (backend em JavaScript)** foi desenvolvida por mim, incluindo:

- Manipulação de dados do carrinho
- Cálculo do total
- Controle das ações dos botões (Adicionar / Limpar)
